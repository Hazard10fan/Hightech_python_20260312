* 파이썬 주요 특징
** 인터프리터 언어(스크립트 언어) : 원시코드를 기계어로 변환안하고 줄(라인)단위로 즉시 명령어 실행
** 동적 타입 언어 : 하나의 변수가 여러가지 자료유형을 다룰수 있는 언어(유연성)
** 플랫폼 독립적 언어 : 대부분 운영체계에서 모두 동작함(윈도우,리눅스,맥등)

* 파이썬 코드 작성시 유의사항
** 파이썬 프로그래밍시 소문자,대문자,숫자,_(underscore)로 조합가능(대소문자 구분함)
  -> underscore는 남발하지않고 최대한 자제해서쓰기(프로그래밍에 혼란을 줄 수도 있음)
** 숫자로 시작해서는 안됨(위반 1var / 가능 var1)
** 키워드(파이썬문법의 식별자)를 식별자 사용x(Global = 10, if = 5)
*** 식별자 : 무언가를 다른것과 구분해서 부를수있는 변수
** 특정심벌 사용x(!,@,#,$,%등)
** 식별자 길이 상관x(추천하는 방식(단어 앞자리마다 대문자섞어서) : Ex. ThisIsALongVariable)
